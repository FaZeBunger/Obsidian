The cost, in machine learning, is some value in machine learning that tells the computer whether it did a good or bad job with it's prediction. The cost is small when the network confidently classifies this image correctly but large when it doesn't know what it's doing. 

The cost of a prediction is given by [[The Cost Function]]. There are many types of cost functions, such as [[Mean Squared Algorithm]] (MSE)
