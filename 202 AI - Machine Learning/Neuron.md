### Description

Think of a neuron as just "some thing that holds a number." Specifically, a number between 0.0 and 1.0 [[Neural Network|Neural Networks]] are just a bunch of neurons connected together. The number inside a neuron is called the "activation" of that neuron.

![[Pasted image 20250428011136.png]]

All information that passes through a neural network must be broken down into some number between 0.0 and 1.0 so that the information can then be stored in these neurons.
