#Psychology/Unit2 


**Error of Omission**: Memory errors where you can't remember something
**Error of Commission**: Memory errors where unwanted information is brought up instead. 

**Memory Decay**: The theory that memories decay over time.


# Error of Omission

**Retroactive Interference**: When newly learned information makes it more difficult to remember old information.

**Proactive Interference**:  When old information interferes with new information.


# Errors of Commission 

**Misattribution**: When we incorrectly recall the source of the information we are trying to remember.

**Deja vu** is a good example of an error of commission. 

**Flashbulb Memories**: Memories for the details surrounding events that are both surprising and particularly significant.
- These are susceptible to errors

**Misattribution effect**: A phenomenon when misleading information alters memory, typically in the context of an eye witness event.
