# Depth of Processing

The best way to learn and form long term memories is not to repeat information over and over again, but instead to **elaborate**.

**Elaboration or Elaborative Rehearsal** reverse to the process of actively manipulating information in working memory so that we can meaningfully connect it to other information we already have in long term memory.

**Deep Processing** is encoding new information by making meaningful connections to existing knowledge. 

**Shallow Processing** is encoding information based on only its surface characteristics.
