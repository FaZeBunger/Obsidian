#Psychology/Unit2 

Long Term Memory is organized into two categories explicit memory, which is conscious, and implicit memory, which is subconscious.

# Explicit Memory

Explicit Memory is further divided into **Episodic** and **Semantic**

## Episodic

Autobiographical memories that are based on life events. These events can be very elaborate or very simple, such as a memory of a birthday party or a memory of what you ate last night.

## Semantic

Information stored that are facts about the world, things that you are told and taught. Facts or figures, things you would learn in school.

# Implicit Memory

## Procedural 

This is information about how a task is to be completed. Such as how to tie a shoe or driving a car.

This is implicit because it involves automaticity. 

## Classical Conditioning

This is implicit because the new association created from [[Classical Conditioning]] becomes automatic.
