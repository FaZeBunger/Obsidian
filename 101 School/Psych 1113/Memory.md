#Psychology/Unit2 

Memory is split into 3 different parts
- [[Encoding Memories|Encoding]]: information acquired and translated into signals the brain can understand.
- Storage: this is where the information is stored for later. We have multiple storage systems such as short term and long term memory. 
- Retrieval: When we search for memory in order to bring information to the surface of consciousness. 

Memory is not infallible. You can remember things incorrectly, it doesn't function like a recording. There is no such thing as a photographic memory, all documented cases were in children and they lost the condition with age. 

**Memory** is a reconstruction.

Memories aren't stored in one place, they're stored all throughout the brain. When you remember something you grab it from many parts of the brain that then have to be re-assembled like a jigsaw puzzle, which introduces error into our memory.
