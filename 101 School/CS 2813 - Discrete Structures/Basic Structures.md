# Sets

**Set:** An unordered collection of distinct objects.
**Element:** One object in this collection

The set O of odd positive integers less than 10 can be expressed by:
- O = {1, 3, 5, 7, 9}.

**Interval:** Sets of all the real numbers between two numbers **a** and **b** 

**Empty Set:** No elements
**Singleton set:** Only element is the empty set


Every nonempty set S is guaranteed to have at least two subsets, the **empty set** and the **set S** itself.

# Set Operations

**Union:** Merging two sets
**Intersection:** The values shared by both **A** and **B**
**Difference:** **A - B** - Remove the intersection of **A & B** from **A.**


# Size of a Set and Power Sets

- **Cardinality** Number of elements in a set.
	- Finite Set -> size |S| = n.
	- Empty Set -> 0.
	- Infinite Set -> Size not finite.

- **Power Set:** All possible subsets (All possible combinations)
	- P(S) = set of all subsets of S.
	- Example:
		- S = {a, b} -> { {}, {a}, {b}, {a, b}}


# Cartesian Products

- **Cartesian Products:** A x B = {$(a, b) | a \in A, b \in B$}

- **Same cardinality:** Exists a bijection (one-to-one and onto).
	- (Perfect Pairing, no leftovers)

- **Smaller / Equal cardinality**: Exists an injection. (A can be embedded into B)

- **Countable**: Finite OR can be listed as a sequence.
